FLAVOUR KINGS

- Logo presentation -

Introduction

Welcome to the Flavour Kings logo samples!

Below are sets of different logo samples. The samples contain variations in typography, colour, and design. Please note that the samples can be tailored and tweaked to your needs and are not necessarily the final result. For example, if you want the braai grid from set 1 placed in set 3, that change can be arranged.

Any changes that you request must be well defined and listed. Please feel free to contact I'MPOSSIBLE if you have any concerns or changes to implement.

SET 1

"Flavour Kings lifestyle lounge"

This set of samples matches the specifications and drawings provided. The typography contains, within the word "flavour", a piece of steak that takes on the role of the letter "A". The letter "U" in lounge is replaced by a Martini glass. The colour scheme is warm, using reds, oranges, yellows, and browns.

SET 2

"Flavour Kings lifestyle lounge"

This set is almost the same as set 1, the difference being the style of steak depicted on the braai as well as the braai itself. This style of steak matches the one in the typography "Flavour Kings".

SET 3

"Flavour Kings lifestyle lounge"

Set 3 is styled after Set 1 and the drawing provided. Set 3 uses a cooked steak and a broad braai grid with sharp corners to stand out more than set 1.

SET 4

"Flavour Kings lifestyle lounge"

In this set I used two fonts that compliment each other and add a sense of class to the logo and brand.

A strong, bold font presents the "Flavour Kings" typography as Kings are seen to be strong and bold. A more relaxing and flowing font is then used for the "lifestyle lounge" typography as lounging is a calm and serene activity. Together, the typographic fonts create a conceptually strong look and feel to the logo and brand.

This set incorporates the requested elements and goes a step further by adding a lounge chair as part of the logo. The braai grid represents the leg extension part of a "lazy-boy" couch, which as a whole represents the lifestyle and lounging aspects of the brand. The logo can also be interpreted as food being served to you as you sit and wait in comfort.

Set 5

"Flavour Kings lifestyle lounge"

In this sample, I used a circular shaped braai as circles represent "wholeness, completion and community." I feel that this concept fits the brand well as people will be looking to commune together at Flavour Kings while they enjoy a drink or eat their meal.

SET 6

"Flavour Kings lifestyle lounge"

Set 6 contains a small variety of options. Some of those options include typography aligned at the bottom of the logo, one cut of steak sizzling "hotter" than the other, and a different way of colouring the typography itself.

SET 7

"Flavour Kings lifestyle lounge"

Set 7 has the typography aligned to the top of the logo and contains small variations, such as the braai grid, steak colour, and font pair.

SET 8

"Chillas meat lounge / Chillas lifestyle lounge"

I included the "Chillas" name in the logo to provide a visual render of how the old name could look with the font choice, colour choice, etc.

SET 9

"Chillas lifestyle lounge"

The above set also uses the "Chillas" name and follows the specifications of the drawings and instructions provided.
